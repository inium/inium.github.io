Computer Graphics의 Real-time Rendering, Image Processing을 했었고 현재는 PHP / Node.js / Docker 등을 이용한 웹 어플리케이션(Frontend / Backend, Database) 개발을 하고 있습니다.

Life

  • 커피와 떼놓을 수 없는 사이입니다. 텀블러에 커피가 항상 있어야 안심이 됩니다.
  • Mac을 사용합니다. 윈도우 노트북을 2년동안 썼는데 결국엔 Mac으로 돌아왔습니다.
  • 담배를 싫어합니다. 술은 알러지 때문에 최대한 절제하고 있습니다.
  • 올빼미 성향인데 다음날을 위해 늦어도 24시에는 잠을 자려 합니다.
  • 집/회사 둘다 해피해킹을 씁니다. vim 모드로 코딩을 합니다.
  • 공부는 생활입니다. 하지 않으면 괜한 불안을 느끼기도 합니다.
  • 체력이 급격히 떨어지는 것을 조금씩 운동과 스트레칭을 합니다. 팔굽혀펴기를 합시다.
  • 가장 하고 싶은 여행은 시베리아 횡단열차 타고 끊임없는 길을 따라 생각에 잠기는 겁니다.
